# Cro-Mag Rally Model Loading

## Model Files

Cro-Mag Rally uses **BG3D format** files located in `/games/cromagrally/models/`.

### Global Models (global.bg3d)
Common objects used across all tracks.

### Vehicle Parts (carparts.bg3d)
Contains car parts and vehicle components.

### Car Selection (carselect.bg3d)
Models for car selection screen.

### Weapons (weapons.bg3d)
Weapon models and projectiles.

### Ramps (ramps.bg3d)
Jump ramps and obstacles.

## Vehicle Types

Cro-Mag Rally features different vehicle types:
1. Stone Wheel - Basic prehistoric car
2. Bone Car - Made of bones
3. Log Car - Hollow log vehicle
4. And more...

## Track Hazards

Each track has unique hazards that are part of the track model:
- Atlantis: Underwater currents, fish
- Egypt: Scarabs, mummy attacks
- Ice: Polar bears, slippery surfaces
- Jungle: Dinosaurs, pterodactyls

## Notes

- Track models contain pre-baked environments
- Vehicle parts are separate for customization
- Some skeleton characters serve as track hazards
