Developer Tools & WASM API

These exported WASM functions can be called from the browser console or injected by your level-editor integration. The game also accepts ?level=N and ?terrainFile=PATH URL params to skip menus and load a specific level on startup.

Terrain Override

Upload a .ter file to inject into the WASM virtual filesystem under /Data/Terrain/. Uses Module.FS.writeFile().

Level Select

Reloads the page with ?level=N to start at this level

Quick Reference

BugdomSetFenceCollision(0) — disable fence collisions (pass through fences)
BugdomSetFenceCollision(1) — re-enable fence collisions
BugdomGetFenceCollision() — returns 1 if fences enabled, 0 if disabled
BugdomGetCurrentLevel() — returns current level number (0–9)
BugdomSetTerrainOverride(path) — override terrain file for next level load

URL params: ?level=3 · ?terrainFile=:Terrain:Custom.ter · ?noFenceCollision=1 · ?dev
