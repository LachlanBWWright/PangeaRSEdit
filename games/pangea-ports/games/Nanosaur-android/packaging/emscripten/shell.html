🎮 Cheat Menu / Level Editor Controls

These controls call exported WASM functions. The game also accepts ?level=N and ?skipMenu=1 URL params.

Cheats & Debug

Terrain Override

Upload a .ter file to inject custom terrain into the WASM virtual filesystem. You can also use FS.writeFile() from the console and then call SetCustomTerrainFile.

Quick Reference

SetFenceCollisionsEnabled(0) — disable fence collisions
SetFenceCollisionsEnabled(1) — re-enable fence collisions
CheatRestoreHealth() — restore health
CheatFillFuel() — fill jet fuel
CheatGetWeapons() — get all weapons
CheatGetAllEggs() — get all eggs
GetGameScore() — returns current score
SetCustomTerrainFile("/Data/Terrain/foo.ter") — override terrain file
ClearCustomTerrainFile() — revert to default terrain
