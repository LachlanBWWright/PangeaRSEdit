JavaScript Cheat / Debug API

Open the browser console while the game is running and use these commands via the gameAPI helper (initialized automatically when the game loads):

// Fence collision
gameAPI.setFenceCollisionEnabled(false);   // disable (walk through fences)
gameAPI.setFenceCollisionEnabled(true);    // re-enable
gameAPI.getFenceCollisionEnabled();         // query (returns true/false)

// Player health and lives
gameAPI.setPlayerHealth(1.0);              // full health (0.0–1.0)
gameAPI.setPlayerLives(9);                 // set lives count
gameAPI.fullHeal();                        // full health + lives + map

// Level control
gameAPI.winLevel();                        // immediately complete current level
gameAPI.setStartLevel(3);                  // queue jump to level 3

Level File Override

Replace any game data file at runtime (useful for testing custom level files):

// Fetch your custom terrain file and write it into the virtual filesystem
fetch('my_custom_level.ter')
  .then(r => r.arrayBuffer())
  .then(buf => {
    var gameFrame = document.getElementById('game-iframe');
    var M = gameFrame.contentWindow.Module;
    M.FS.writeFile('Data/Terrain/Level1_Garden.ter', new Uint8Array(buf));
  });

Call this before the level loads (e.g. right after the game starts with ?level=0). The game will use your file instead of the bundled one.
